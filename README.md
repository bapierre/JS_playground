# JS_playground

## Le but de cette application est de créer une suite de jeux simple tournant sur le navigateur afin de s'entraîner et apprendre :
  - **HTML**
  - **CSS**
  - **JS**
  - **Comment combiner les 3 pour créer un site internet**
  
  
  
## Les jeux actuellement implémentés (ou en cours d'implémentation) sont :
  - **Pong** : Le jeu se joue à 2 sur un même clavier, une personne utilise les touches "z" et "s" pour déplacer sa barre tandis que le second utilise les flèches directionnelles vers le haut et le bas
  - **Snake vs** : Jeu de snake 2 joueurs où le but est de forcer l'adversaire à se mordre la queue ou à mordre la sienne dans le style de Slither.io
